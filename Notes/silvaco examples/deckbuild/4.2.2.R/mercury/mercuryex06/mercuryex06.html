Gain Compression

Requires:Mercury
Minimum Versions:Atlas 5.26.1.R


This example defines a simple, recessed, n-type MESFET and does a Harmonic Balance simulation at 100 MHz and 1 GHz.

This example goes to higher input power than the previous example and so displays the gain compression that occurs.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
