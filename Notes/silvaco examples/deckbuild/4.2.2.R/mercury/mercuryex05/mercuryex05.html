Harmonic Balance

Requires:Mercury
Minimum Versions:Atlas 5.26.1.R


As the Mercury module uses a 1D simulation to calculate the transport, it uses a relatively small number of mesh points (compared to a 2D Atlas simulation of the same FET). Therefore Mercury can use the Harmonic Balance method to simulate the large signal behavior of the FET. The matrices used in the Harmonic Balance method are 25-100 times bigger than the matrices used in a DC simulation of the same device.

This example defines a simple, recessed, n-type MESFET and does a Harmonic Balance simulation at 100 MHz.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
