Trend Analysis

Requires:DBInternal, Mercury
Minimum Versions:Atlas 5.26.1.R


The speed of Mercury means it is well suited for trend analysis, investigating how changes in the structure of the device affect the output.

This example shows how the doping in a device effects the current pinch off behavior.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
