User-defined scattering example

Requires:Mocasim, TonyPlot
Minimum Versions:Mocasim 5.16.0.R

This example demonstrates the bulk parameter simulation of a default material (GaAs), however user-defined scattering mechanisms are defined for the Gamma<->L intervalley transition.

This example is basically the same as example 1 except that some of the scattering mechanisms are defined by a c-interpreter file (and we don't fit a model to the results, so there is no Section 7 in this example).

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
