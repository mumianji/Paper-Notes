Dose Dependent Channeling of Phosphorus

Requires:SSuprem 4
Minimum Versions:Athena 5.22.3.R


This example shows the dose dependence of zero-tilt P implant profiles obtained using the SIMS Verified Dual Pearson ( SVDP ) method and compares them with experimental results (R.J. Shreutelkamp, et.al., Nucl.Instr. & Meth., B55, p.615, 1991) saved in *.exp files. As the implant dose increases, more damage is caused resulting in additional dechanneling of phosphorus ions. Therefore, the profile tail shortens with increasing dose.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
