Substrate Shear Stress

Requires:SSuprem 4
Minimum Versions:Athena 5.22.3.R


This example demonstrates the capability to calculate shear stress as a result of an overlying nitride layer. The results of the calculation are plotted using TonyPlot to display contours of each of the three components of stress produced by the calculation.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
