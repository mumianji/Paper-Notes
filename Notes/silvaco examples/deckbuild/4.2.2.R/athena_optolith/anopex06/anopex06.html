Aberrated Aerial Image

Requires:SSuprem 4/Optolith
Minimum Versions:Athena 5.22.3.R


This example simulates an aerial image for a projection system that has 0.5 lambda "astigmatism", and the position of the image patch under consideration is located at the upper right boundary of the circular image field.

The input deck is composed of two parts: simulation of the aerial image without and with aberrations (astigmatism).

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
