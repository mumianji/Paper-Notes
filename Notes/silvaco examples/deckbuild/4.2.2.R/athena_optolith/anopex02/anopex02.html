Arbitrary Source Shapes

Requires:SSuprem 4/Optolith
Minimum Versions:Athena 5.22.3.R


This example is a comparison of imaging using a SHRINC source with diagonal and perpendicular orientation of individual sources. The source aerial images are calculated and compared with each other. Also, maximum normal intensities for both cases are extracted.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
