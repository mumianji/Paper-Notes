Bipolar Example Using MaskViews

Requires:SSuprem 4, MaskViews
Minimum Versions:Athena 5.22.3.R


NOTE:

This example works only on UNIX/LINUX computers.

This example uses an IC Layout cross section together with a generic process flow to realize a specific device.

To run this example, first invoke MaskViews from the Tools - MaskViews - Start MaskViews options. Create a cutline object by selecting the Write file button, and then selecting the beginning and end of the required cross section. Load the cutline object into DeckBuild by using the Tools - MaskViews - Cutlines menu option. Once loaded into DeckBuild , press run .

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
