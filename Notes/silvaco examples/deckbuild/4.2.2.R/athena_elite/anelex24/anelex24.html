Customizable Deposit from an ASCII File

Requires:Elite
Minimum Versions:Athena 5.22.3.R


This example shows the use of Elite to model a customizable deposit. Input is performed via an ASCII file containing angle and rate information.

The example begins by creating a test structure using geometrical etch with a START CONT DONE sequence of ETCH statements. This creates a globe on which deposition will be performed and compared. The globe is then deposited with rates that are included in a data file using the CUSTOM1 model. The first data file anelex24_1.dat specifies that the deposit rates will be constant at all angles except at a single angle (-45 degrees) where the rate is ten times higher.

A second data file anelex24_2.dat specifies cosine rate distribution with a maximum rate in the vertical direction. Finally, the deposit is repeated using a standard unidirectional model.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
