Oxidation Induced Stress Enhanced Etch Rate

Requires:SSUPREM4, Elite, MC DEPOSIT/ETCH
Minimum Versions:Athena 5.22.3.R


This example demonstrates the stress enhanced etching using the Elite module of Athena. A simple trench is etched into the silicon substrate through a region near a bird's beak. This region contains two high Sxy stress areas developed after oxidation. Oxidation is simulated using a stress-dependent viscous model.

A standard RIE etching machine is defined using the RATE.ETCH statement. Stress enhancement of the etch rate is specified in the RATE.DOPE statement.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
