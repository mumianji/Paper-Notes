Unidirectional Deposit Example

Requires:Elite
Minimum Versions:Athena 5.22.3.R


This example uses the unidirectional deposit model with incident angles of 0, 45, -45, and 80 degrees from the surface normal. The unidirectional model uses a cosine law deposit that is applicable to line of sight deposition.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
