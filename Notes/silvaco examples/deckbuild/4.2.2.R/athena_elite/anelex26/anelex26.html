The Soft Polish Model

Requires:Elite
Minimum Versions:Athena 5.22.3.R


This example shows the use of Elite to model chemical mechanical polishing using the soft polish model. This example calculates results for the the soft polish model so that they can be compared to the published results by J. Warnock, "A Two Dimensional Process Model for Chemimechanical Polish Planarization", Electrochem. Soc. Vol 138, No. 8, August 1991.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
