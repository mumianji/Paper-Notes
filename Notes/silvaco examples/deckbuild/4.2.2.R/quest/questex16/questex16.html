Parameterized MIM Capacitor layout

Requires:QUEST
Minimum Versions:QUEST 2.0.2.R

This example demonstrates the use of an Expert LISA script used by QUEST to generate a parametrized MIM Capacitor gds layout.

Introduction:

In order to build an MIM capacitor model or simply make variation of layout parameters it is especially interesting to have parametrized gds. This is possible in QUEST by using the Expert layout editor's LISA scripting language.

Running the Example

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.

In this example, two layout variables are defined (Width, Length) and the layout is created accordingly using the creategds command of QUEST associated with the Expert script file.

Results analysis

Inductor performance is monitored using the extract capability in DeckBuild.
