Mixed Ambient Oxidation

Requires:SSuprem 4
Minimum Versions:Athena 5.22.3.R


This example demonstrates SSuprem 4's ability to perform oxidation in mixed gas ambients. The gas flows can be specified in relative units for each flow of oxygen, water, hydrogen, and nitrogen by the parameters F.O2, F.H2O, F.H2, and F.N2 on the DIFFUSE statement.

For this example, the oxidation step is performed several times using SSuprem 4's internal looping capability to specify a different gas mixture each time the loop is executed.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
