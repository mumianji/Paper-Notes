Controlling Encroachment using Oxidation Threshold

Requires:SSuprem 4
Minimum Versions:Athena 5.22.3.R


This example specifies that oxidation only occurs when oxidant concentration is above a certain value.

Two oxidation threshold values are used in this example along with a run using the default model. The threshold model is enabled by: method ox.thresh

Parameters for the model are set by material and separately for wet and dry oxidations using the statement oxide min.oxidant=<value>

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
