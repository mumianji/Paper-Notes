Void Formation During Trench Oxidation

Requires:SSuprem 4
Minimum Versions:Athena 5.22.3.R


This example demonstrates SSuprem 4's ability to oxidize when colliding oxide fronts cause a void. A narrow trench is oxidized until the opposing oxidation fronts touch. During the oxidation SSuprem 4 saves the structure after each time step. These saved structures as well as the final structure are used to monitor the moving boundary progress.

To watch the movie, first select "Select all plots" from the "View" popup and then select "Movie.." from the "Tools" popup.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
