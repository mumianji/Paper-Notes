A 3D Elliptical Source with a Gaussian Optical Intensity

Requires:Device 3D/Luminous 3D
Minimum Versions:Atlas 5.26.1.R


This example demonstrates:

  • How to define a 3D elliptical source with a Gaussian optical intensity profile.

This example combines the example optoex11.in with example optoex10.in to create a gaussian optical intensity profile for a 3D optical source. A description of the syntax for defining these effects is decribed in those examples.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
