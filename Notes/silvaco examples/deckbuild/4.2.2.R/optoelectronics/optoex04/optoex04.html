AC Response

Requires:S-Pisces/Luminous
Minimum Versions:Atlas 5.26.1.R


This input file analyzes the small signal response of the device to a small signal optical source.

The first four sections of the input file are identical to the corresponding sections in the input file for the DC example described previously.

The fifth section of the file specifies that a DC solution at a source intensity of 5 W/cm^2 is obtained. After the DC solution, the small signal response of the device to an small signal optical source of 1 mW/cm^2 is extracted for frequencies ranging from 100KHz to 100GHz. The full AC terminal characteristics are saved in the log file.

When the results are viewed in TonyPlot, they show that the device has a cutoff frequency of about 1 GHz.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
