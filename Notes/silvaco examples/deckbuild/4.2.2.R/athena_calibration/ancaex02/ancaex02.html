OED Calibration Using THETA.0

Requires:SSuprem 4
Minimum Versions:Athena 5.22.3.R


This example shows how to use the interstitial injection coefficient theta.0 for calibration of Oxidation Enhanced Diffusion. Only prefactor theta.0 is used in this example. If data on temperature dependence of OED was available, theta.E could be calibrated as well.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
