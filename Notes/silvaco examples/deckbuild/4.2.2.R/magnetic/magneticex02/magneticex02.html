Current based Magnetic field detector

Requires:S-Pisces/Magnetic
Minimum Versions:Atlas 5.26.1.R


The example shows :

  • A dual collector bipolar structure made using Atlas syntax
  • Bifurcation of the collector currents due to Magnetic field

Without the field, the collector currents are identical, applying the magnetic field breaks the symmetry causing more current to flow to one of the collectors. The difference in collector currents allows the magnetic field to be measured. This structure is not strictly speaking a magneto-transistor as the base thickness is zero in some directions.

To load and run this example, select the Load button in DeckBuild > Examples. This will copy the input file and any support files to your current working directory. Select the Run button in DeckBuild to execute the example.
